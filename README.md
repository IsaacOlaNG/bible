# Bible
Read the Bible with python and curses.
![Preview](preview.png)

## Installation
```shell
$ pip install https://github.com/rwev/bible/archive/master.zip
```

## Usage 
```shell
$ bible
```
Use the arrow keys to navigate and make selections.
